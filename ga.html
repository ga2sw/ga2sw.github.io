Get Client ID, Client Secret and Developer API Key for Google Developers Console Application

Following are the simple steps which can help you to get your Client ID, Client Secret and Developer API Key for your Web Apps.

  1. You need to create a Project in Google Developer Console. Click here to go to Console page. It will ask you to login with your Google Account if you are not logged in already. If you have created Projects in Google Developers Console before you will see list of already created projects. If not, below screen will appear. Start – Create Project in Google Developers Console
  2. In both scenarios, Click on ‘Create Project’ button.
  3. A popup will appear, where you will enter ‘Project Name’ and ‘Project ID’ which will help you to recognize your project in console.
    Enter Project Name – Google Developers Console
  4. By clicking on Create button, Your project will be created and redirects you to your projects list in Console.
  5. Clicking on the project name which you have recently created will take you to your Project Dashboard.
  6. See left side navigation menu. Go to APIs & auth -> APIs and Select ‘Analytics API’ and Turn in ‘ON’.
    Enable Ananlytics API in Google Developers Console
  7. Next, Go to APIs & Auth -> Credentials -> OAuth Consent Screen and Enter your Product name “My-GA2SW” and Save it.
    Enter Product Name – Google Developers Console
  8. Go to APIs & auth -> Credentials. Click on OAuth 2.0 Client ID and Select ‘Website’ from options. Ensure to add the Authorized Redirect Urls
    Get OAuth 2.0 Client ID – Google Developers Console
  9. You will see a popup containing Client ID and Client Secret. Copy that and paste it in Plugin.
  10. Now, you need to get Public API Key.
    Create Public API Key – Google Project Console
  11. Click the Browser Key and press ‘Create‘ button
    Select Browser Key – Google Project Console
  12. If you have done all the above steps correctly, you should see the page like below screen.
    Get your Client ID, Client Secret and API keys – Google Project Console.
