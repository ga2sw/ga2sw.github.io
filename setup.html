Deploying to Heroku using the Command Line

You can also deploy GA2SW to Heroku using the command line:

  1. Clone the repository

    git clone https://github.com/Cervello/google-analytics-to-salesforce-wave

  2. Create a Heroku application

    cd ga2sw 
    heroku create

  3. Install the Postgres plugin 

    heroku addons:add heroku-postgresql:dev

  4. Deploy to Heroku

    git push heroku master

  5. Run the the application:

    • Open the application in a browser:

      heroku open

    • Use "Admin" as username and "admin" as password to login

Installing a Local Version

You can also install GA2SW on your local machine:

  1. Clone the repository

    git clone https://github.com/Cervello/google-analytics-to-salesforce-wave

  2. Install the server dependencies

    cd ga2sw 
    npm install

  3. Create a local database

    • Install and start Postgres on your local machine
    • Create a database called ga2sw
    • If your database is available using postgres://@127.0.0.1:5432/ga2sw, you have nothing else to do

  4. Start the server 

    node server

  5. Run the application

    • Open a browser and access the following URL: http://localhost:5000
    • Use "Admin" as username and "admin" as password to login
